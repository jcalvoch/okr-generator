# OKR Generator

A simple Next.js app to create and preview OKRs: one objective, two key results, three KPIs, and a reasoning box.

## Setup

From the `okr-app` folder:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## What you get

- **Input form**: Objective, period, Key Result 1 & 2 (description + Actual / Meta / Inicio), three KPIs (title, value, trend), and a **Reasoning** text area for the OKR decision.
- **Dashboard preview**: Live preview below the form matching your inputs (cards for objective, KRs with progress bars, KPIs with mini charts, and the reasoning card at the bottom).

No extra dependencies beyond the default Next.js stack.
